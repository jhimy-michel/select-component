## Setup
1. Clone this repo
2. Install all the dependencies (npm install)
3. Run in a terminal: `npm run dev`

## Deploy a new version
 Navigate to the scripts folder:

 ```bash
 cd scripts/
 ```

 And execute the `deploy.sh` script:

 ```bash
 ./deploy.sh
 ```
